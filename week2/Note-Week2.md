$\mathbb{P}$ = Probability
### $\mathbb{P}(Event) :$
- $\mathbb{P}(A),$
- $\mathbb{P}(B),$
- $\mathbb{P}(A\cap B),$
- $\mathbb{P}(A\cup B)$
### Example
Coin
A=Head
B=Tail

$A \cap B = \phi$ (The coin is head and tail, Not posible)

$A\cup B$ = $\Omega$ (The coin is hear **or** tail)

$\mathbb{P}(\Omega)$ =1
$\mathbb{P}(\phi)$ = 0

$\phi \cup \phi = \phi$

$F =$ { $\phi, \Omega, A, B$ }

$S = \Omega$

$\Omega \in \mathcal{F}$
$F \in \mathcal{F}$
$F,G \in \mathcal{F}$

$F\cup G \in \mathcal{F}$
